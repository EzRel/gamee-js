Gamee JS 2
==========

Javascript framework for [Gamee].

[Documentation on github wiki] 

Previous versions
-----------------

[Gamee JS - first version]

Install & build
---------------

```
git clone git@github.com:gameeapp/gamee-js.git
```

```bash
npm install
bower install
grunt
grunt watch (Framework development) 
```
